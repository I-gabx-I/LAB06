# LAB06
# Task Manager 📝

Gestor de tareas con **React** y **TypeScript**.

## 🚀 Características
- Agregar, editar y eliminar tareas.
- Filtrar por categoría.
- Diseño mejorado con estilos modernos.

## 📌 Instalación
```sh
npm install
npm start
```
O con **yarn**:
```sh
yarn install
yarn start
```

## 📁 Estructura
```
/src
  ├── components
  ├── App.tsx
  ├── index.tsx
  ├── styles.css
```
